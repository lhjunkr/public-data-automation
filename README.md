# Public Data Automation

공공정보 전달 격차 해소를 목표로 하는 자동화 플랫폼입니다.

이 프로젝트는 대구 지역 공공정보를 RSS/API로 수집하고, AI로 시민이 이해하기
쉬운 콘텐츠로 변환한 뒤 Facebook, Instagram, Threads, Naver Band 등으로 자동
배포하는 것을 목표로 합니다.

## Initial Scope

첫 번째 구현 대상은 `대구 채용·시험` 정보입니다.

### 공공 부문 RSS

- 공채/경채 공무원
- 임기제/별정직 공무원
- 청원경찰
- 개방형직위
- 중앙 및 타기관 채용소식

### 민간 부문 API

- 워크넷 채용정보 API

## Development Status

새로 시작하는 저장소입니다. 기존 `meta_automation` 코드를 그대로 옮기지 않고,
공공정보 도메인에 맞는 구조로 다시 설계합니다.
